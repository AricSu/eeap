# EEAP Spec

This directory contains the normative EEAP specification documents.

## V0 Priority

The first milestone is the EEAP core:

- object model
- lifecycle semantics
- evidence semantics
- idempotency expectations

These are intentionally defined before:

- transport bindings
- authorization profiles
- discovery metadata
- vendor-specific connector patterns

## Documents

- `architecture.md`: system boundary, flow, and repository structure
- `core.md`: canonical objects and field-level intent
- `lifecycle.md`: event semantics and lifecycle rules
- `positioning.md`: comparison against adjacent standards and protocols
